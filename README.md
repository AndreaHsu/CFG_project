# CFG
## 使用技術
    - python
    - Jieba斷詞
    - WordCloud生成文字雲
## 技術文件
    - craw_Dcard.py 搜尋關鍵字，爬取Dcard文章
    - craw_PTT.py 搜尋關鍵字，爬取PTT文章
    - cut_word.py 斷詞並儲存成.dataset檔
    - textRank.py 抽取每篇文章的前4重要文句作為參考
    - wordCloud.py 生成文字雲
    - Data資料夾放置所有爬蟲的原始資料、斷詞處理完的資料、textRank(文句重要排序)的文件
    - CFG.jpg是wordCloud生成的文字雲
